# dailz-go
Pet project backend
Задача проекта:
1. Запустить БД Postgres SQL
2. Написать Модель Условного юзера, его данных
3. Написать функцию, при вызове которой юзер записывается в БД
4. Написать тест, который будет тестировать запись юзера в БД. И Юнит, который будет тестировать функцию(если это возможно)
